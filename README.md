5. If your frames reach the server it will send ACK signal to client
6. Stop the Program
7. 
## PROGRAM :

### CLIENT :

```
NAME: SANJITH R
REG NO: 212223230191

import socket
s=socket.socket()
s.bind(('localhost',8000))
s.listen(5)
c,addr=s.accept()
while True:
    i=input("Enter a data: ")
    c.send(i.encode())
    ack=c.recv(1024).decode()
    if ack:
        print(ack)
        continue
    else:
        c.close()
        break

```
